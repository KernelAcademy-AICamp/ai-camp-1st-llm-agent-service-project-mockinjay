Accessibility Guidelines

  • ✓ All text colors meet WCAG 2.2 Level AA (4.5:1 contrast minimum)
  • ✓ Large text and UI elements meet 3:1 contrast minimum
  • ✓ Color-blind safe when used with icons and text labels
  • ✓ High contrast mode automatically supported
  • ✓ Forced colors mode (Windows High Contrast) supported
  • ⚠️ Always add icons to semantic colors (never rely on color alone)
  • ⚠️ Test with simulators for protanopia, deuteranopia, tritanopia
