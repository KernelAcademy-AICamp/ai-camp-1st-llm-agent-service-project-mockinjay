기능 ID: UTI-001
대분류: 공통
중분류: 유틸리티
소분류: Header
기능명: Header
기능 상세: 서비스 로고, 타이틀, 네비게이션 메뉴(Home, Knowledge Search, NutriCoach, Quiz, Community), 로그인/로그아웃 버튼, 사용자 프로필 아이콘, 알림 아이콘(새 알림 Badge 표시) 구성

우선순위: P0
적용: O

중분류: 유틸리티
소분류: Side Bar
기능명: Side Bar
기능 상세: PC 환경에서는 좌측에 고정 사이드바 표시, Mobile 환경에서는 햄버거 메뉴(...) 아이콘 클릭 시 슬라이드 형태로 표시. 각 메뉴 항목 클릭 시 해당 페이지로 이동

출력: 사이드바 메뉴: Home, Knowledge Search, NutriCoach, Community, MyPage. 현재 페이지 하이라이트 표시
우선순위: P0
적용: X

기능 ID: UTI-004
대분류: 공통
중분류: 유틸리티
소분류: Notification
기능명: 알림 기능
기능 상세: Header의 알림 아이콘(🔔)에 새 알림 개수 Badge 표시. 알림 아이콘 클릭 시 알림 목록 드롭다운 표시. 알림 유형: 업데이트 공지, 커뮤니티 댓글/좋아요, 설문 조사 마감 임박, 챌린지 참여 독려, 퀴즈 알림, 레벨업 축하. 각 알림 클릭 시 관련 페이지로 이동. 알림 읽음 처리 및 전체 삭제 기능 제공
입력: 알림 아이콘 클릭, 새 알림 발생 이벤트
처리: 새 알림 발생 시 → DB에 알림 저장(사용자 ID, 알림 유형, 메시지, 읽음 여부, 생성 시간, 관련 링크) → 알림 아이콘에 미확인 개수 Badge 업데이트. 알림 아이콘 클릭 시 → 알림 목록 조회(최근 20개) → 드롭다운 표시. 알림 클릭 시 → 읽음 처리(DB 업데이트) → 관련 페이지로 이동. 전체 삭제 클릭 시 → 모든 알림 삭제
출력: 알림 아이콘 Badge(미확인 개수) + 알림 목록 드롭다운(알림 제목, 시간, 읽음/안읽음 표시) + 각 알림 클릭 시 관련 페이지 이동 + "전체 삭제" 버튼
우선순위: P2

기능 ID: UTI-005
대분류: 공통
중분류: 유틸리티
소분류: System Error
기능명: System Error Message
기능 상세: HTTP 404(페이지를 찾을 수 없음), 500(서버 내부 오류), 502(게이트웨이 오류) 발생 시 사용자 친화적인 오류 메시지와 홈으로 돌아가기 버튼 표시
입력: HTTP 에러 코드(404, 500, 502)
처리: 백엔드에서 에러 응답 → 에러 코드에 따른 메시지 매핑 → 에러 페이지 렌더링
출력: 에러 페이지: 에러 코드, 사용자 친화적 메시지("페이지를 찾을 수 없습니다" 등), 홈으로 돌아가기 버튼
우선순위: P0
